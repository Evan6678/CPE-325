CPE 325 - Final Project


Description:

This app is a basic 3x3 matrix calculator using the MSP430FG4618 microcontroller. It is written in C and assembly and utilizes UART for terminal comminication and the MSP430 accumulator for multiplication.

Setup:

This section will explain how to setup and run the project. First you will need to create a project in TI Code Composer Studio and configure it as we were shown in lab1 (steps provided in link below). Next you will take the provided main.c and MULT.asm files and save them into your project. Third you will open Mobaxterm or Putty and configure your serial port as COM1 at a baud rate of 19200. Run the project making sure your MPS430 has power and is connected to the serial port.

Configure Steps

Instructions:

The project can be run by only using numbers and the SPACE bar. For each position in your matrix insert a desired number between 0-9,999 then press the SPACE bar. The SPACE bar will automatically save your input and prepare you for the next input. It will also determine where in the matrix you are and will automatically do things for you such as move to the next row, move to the next matrix input, and generate all your calculations after all the inputs are made.

Constraints:

Due to limited time for this project some constraints had to be made to limit the scope and complexity.

  • Negative inputs are not allowed
  • Values greater than 9,999 are not allowed
  • Backspace to delete a number input does not work
  • 3x3 matix is the only size allowed
